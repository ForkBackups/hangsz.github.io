研究生阶段工作总结

发表： 2017-11-20 | 更新： 2016-12-04 | 专题： 大学 | | 阅读次数：
目标：写出能用于旋翼（Rotor）飞行器流体模拟计算的软件
2016年6月1日 儿童节 与师弟合作下，目标达成。

主要就是写了个软件，用之做了点研究。
专栏：流场求解器初探
交流学习：hangzs@nuaa.edu.cn

软件简介

1. RotorAS-3D

2016.6 完成，全速域3D版本，最终代码量2万行左右。
包含以下功能：

  1. 前处理-结构网格生成、非结构网格生成以及混合网格生成
    结构网格：代数法法向推进，泊松方程光顺；非结构网格：阵面推进法
  2. 求解器—基于密度，可用于全速域流场求解计算
    空间离散：JST、Roe格式；时间离散：统一采用双时间步法，隐式推进；离散方程组求解方法：LU-SGS
    湍流模型：SA 和 k-omg SST
    不可压缩计算采用预处理技术，动态计算采用非结构网格的嵌套网格技术和动网格技术
  3. 后处理——模拟数据处理与可视化
  4. GUI设设计
2. RotorAS-2D

20.16.2 完成，全速域2D版本。

3.RotorC-2D

2016.2 完成，可压缩2D版本，功能比 RotorAS-2D版本弱一些，计算不可压由于特征值量级差异导致收敛困难

软件开发者：孙振航 桑树浩
单位：旋翼动力学国家级重点实验室
