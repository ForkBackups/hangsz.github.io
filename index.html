win下配置OpenGL环境（sb6）

发表于 2016-03-26 | 分类于 Computer Graphics

OpenGL 配置方法（先自行安装vs2013）

  1. glew下载1.7.0版本（win32 or 64）

    将所有压缩包中 include 文件夹下的 .h 文件拷贝到 D:\Program Files\Microsoft Visual Studio 12.0\VC\include\GL目录中（若没有GL文件夹请自行创建，VS目录不一定相同）。
    将lib文件夹下的 .lib 文件拷贝到D:\Program Files\Microsoft Visual Studio 12.0\VC\lib中。
    将 bin 文件夹下的 .dll 文件拷贝到C:\Windows\System32目录（win32），或者C:\Windows\SysWOW64目录（win64）。

  2. glut下载3.7版本

    glut文件夹下的 .h .lib 和.dll ，处理方法同上。

  3. glfw 下载2.7.6版本

    只要将 include中的 .h 文件按照上述方法处理就可以了。
