写完毕业论文后，闲来无事，准备梳理一下脑中的数学知识，能够建立更好的知识体系是我的目标，如果对试图厘清数分思路的同学有所帮助也是极好的。本来预计要写一篇文章的，结果写起来发现一篇根本hold不住，那么暂且先拆分一下。

我眼中的数学

数学不是自然科学，类似于宗教可以自行演化一个世界。现代数学基于$ZFC$（策梅洛-弗兰克尔集合论）公理系统，通过定义概念、定义运算，采用演绎推理获得了一些有意义的引理、命题、定理等，这些定理浓缩着智慧，即可指导实践也可为深入研究铺路。

在计算机科学领域，有一句话叫没有什么是加一层解决不了的问题。在我看来，数学就是现实与思维之间的间接层，现实实例很多——一棵树、一个人、一本书，一棵树 “加”一个树是两个树……如果需要人脑对每种实例都建立一套描述体系，那么还不把人累死。所以抽象出了 自然数 $1$ 用来描述所有实例的个数概念，抽象出了 $+$ 来表示数的相加。自然数系的出现只是简单的抽象，而数学体系内部会有现有概念的更高层、更复杂的抽象。不过就像coding一样，并不是接口层越抽象越好，数学本身内部的更高层级抽象也难说好坏，更多思考可以看这个问题什么是好的数学。

回到工科必修的数学分析上来，在我眼里数学分析就研究了两个对象 1. 集合 2.映射，具体说来就是：

  • 集合的结构和性质，结构如元素构成，性质如可数性、有界性、完备性等。
  • 映射的结构与性质，结构如XX类型集合到XX类型集合，性质如连续性、可微性等。

这两者既相互独立又相互交织，其实线性代数也是如此，研究对象只有线性空间（集合）和线性变换（映射）。

1. 集合的结构与性质

集合的抽象层次太高，而工科需要的数学分析要与实际结合，能解决实际工程问题，所以数学分析具体讲的是两个具体的集合对象—数集和欧几里得空间，其是工程问题的直接抽象，相信大家对这一点都深有感触。

  • 数集（$N$、$R$等），从满足Peano公理的自然数集到通过有理数柯西序列定义的实数集，数集不断扩充，其性质也变得更为丰富。
  • 欧几里得空间（$R^{n}$ ），形如$(x_{1}, …,x_{n})$的有序元组构成的集合，多元函数微积分依赖的集合。

数集和欧几里得空间更高层的统一概念是度量空间，也即定义了距离函数（度量）的集合。对于集合$X$，度量定义为$d:=X\times X\rightarrow [0,\infty )$，换句话说对于任意$x,y\in X$，有$d(x,y)\geq 0$且$d$要满足正性、对称性和三角不等式等性质。对于度量空间还有更高层的拓扑空间抽象，这里就不展开了，也用不到。

数集上的标准度量是绝对值$ d(x,y)=\left| x-y \right|$ 。欧几里得空间的度量常用的有3种：
$l^{2}$ 度量，$d_{l^{2} } ((x_{1},…,x_{n}),(y_{1},…,y_{n} ))=\sqrt{(x_{1}-y_{1} )^{2} +…+(x_{n}-y_{n} )^{2} } $
$l^{1}$度量，$d_{l^{1} } ((x_{1},…,x_{n}),(y_{1},…,y_{n} ))=\left| x_{1}-y_{1} \right| +…+\left| x_{n}-y_{n} \right|$
$l^{\infty }$ 度量，$d_{l^{\infty } } ((x_{1},…,x_{n}),(y_{1},…,y_{n} ))= sup\left\{ \left| x_{i}-y_{i} \right| :1 \leq i\leq n \right\} $
有了度量就可以定义更多的概念，下面只谈与映射不相关的概念，其是后续映射研究的基础。比较重要的概念包括：

  • 附着点，内点、外点、边界点等概念，通过度量定义的；
  • 开集和闭集，如数域上区间，闭集包括所有边界点，开集不包括任何边界点；
  • 有界性，集合是否有界；

2.映射的结构

映射、函数、变换、态射等等都是在不同集合上讨论时用的相同概念，这些概念抽象等级太高，数分里研究的集合就两种，对应的映射类型其实也就以下四种：

  1. 序列（数列），自然数集$N$ 到 实数集$R$的映射；
  2. 一元函数， 实数集$R$ 到 实数集$R$的映射；
  3. 多元函数，欧几里得空间$R^{n} $ 到 实数集$R$的映射；
  4. 函数序列，自然数集$N$ 到 函数空间（定义为集合$X$到集合$Y$的映射构成的集合）的映射。

序列或函数这些映射本身就是对象，也可以定义对象具有的普遍性质，重要的包括：

  • 相等，相等概念需要满足自反、对称、传递和代入四个公理；
  • 运算，重要的有4种： 加减乘除（分母不为零）、数乘 、$max$和$min$、复合。

3. 映射的什么性质值得研究？

有一个数列，那么序列的什么方面值得研究呢？现实中常常有这么一种序列$a_{n}$ ，随着$n$不断增大，两个元素越来越接近，基于这样一种事实，引入了数分乃至整个分析体系的一个很重要的概念——极限。

还记得前面提到了度量概念吗？有了度量才能定义极限，因为极限语言（$\epsilon -N $或 $\epsilon -\delta$ ）表述无限接近这种大白话时需要用度量来严格表述。比如序列$a_{n}$ 的极限定义：对于任意$\epsilon $，都存在一个$N$，使得$n \geq N$时，$a_{n} $都$\epsilon$ 接近于一个数$L$，$\epsilon$ 接近意味着$d(a_{n} ,L)<\epsilon $。此时记作$\lim_{n \rightarrow \infty } {a_{n} }=L $，$L$就是所谓的序列极限。函数也可以采用类似的方式来定义函数极限，其采用的极限语言则是$\epsilon -\delta $。

有些人觉得序列极限和函数极限不同，其实不是的，是一样的东西，函数极限可以转用序列极限语言来表述的。在更高层的集合抽象中，比如拓扑空间，这些形状各异的极限都归结于一种。

如果序列有极限，极限是唯一的。极限表面上是一个对象、一个值，而本质上则是用一个新的概念来表述某个序列无限接近的动态性质。
对于极限这个对象本身，也可以定义对象具有的一些普遍性质：
运算，重要的有3种： 加减乘除、数乘 、$max$和$min$。

至此，引出了数分中的基本概念—极限，后续的级数、微积分和Fourier级数等也都可以逐个亮相了。
